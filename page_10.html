Phase 2 - Design Skills and Tools

Design is about creating the visual elements and user experience of a website. Here are the key skills and tools:

UI Design

Skills Required:

  • Color theory
  • Typography
  • Layout principles
  • Visual hierarchy

Tools:

Figma
Adobe XD
Sketch

UX Design

Skills Required:

  • User research
  • Wireframing
  • Prototyping
  • Usability testing

Tools:

Balsamiq
InVision
Marvel App

Assets

Skills Required:

  • Image editing
  • Vector graphics
  • Icon design

Tools:

Adobe Illustrator
Canva
SVG editors

"UI Design is like decorating a house, while UX Design is like planning how people will move through it."
