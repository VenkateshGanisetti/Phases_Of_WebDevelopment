Phase 5 - Deployment Skills and Tools

Deployment is the phase where your website moves from development to a live server, making it accessible to users via the internet.

Hosting

  • Netlify or Vercel: Easiest for beginners
  • Heroku: Platform as a service
  • GoDaddy: Domain registration

Skills: Domain management, DNS configuration

Cloud Services

  • AWS: Amazon Web Services
  • Google Cloud: Scalable infrastructure
  • Microsoft Azure: Cloud computing platform

Skills: Scalability, file storage

CI/CD

  • GitHub Actions: Workflow automation
  • Jenkins: Open-source automation server
  • Docker: Containerization platform

Skills: Continuous Integration/Deployment

Deployment skills are essential for making your website live and accessible to users worldwide.
