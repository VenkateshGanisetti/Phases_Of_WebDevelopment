Phase 6 - Maintenance Skills and Tools

A website is never truly "finished." It needs ongoing care to remain secure, functional, and relevant.

Analytics

Tracking user behavior and traffic analysis

  • Google Analytics: Tracks user behavior and traffic sources
  • Hotjar: Visual feedback and user session recordings
  • Skill: Data interpretation for site improvement

Support

Bug tracking, ticket management, and customer support

  • Zendesk: Customer support platform
  • Jira Service Desk: Ticket management
  • Skill: Problem-solving and communication

Updates

Security patching, dependency management, content updates

  • NPM (Node Package Manager): Manages dependencies
  • CMS Dashboards (e.g., WordPress): Content management
  • Skill: Version control and security awareness
