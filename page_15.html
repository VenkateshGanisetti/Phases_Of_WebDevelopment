Phase 5 - Deployment

The "Going Live" Phase

The website is moved from a development environment to a live server, making it accessible to the world via the internet.

"You get the keys to your new house and officially move in! You invite your friends and family over for a housewarming party. The house is now "live"."

Upload Code & Databases

The code and databases are uploaded to a web hosting server.

Point Domain Name

The domain name (e.g., www.myawesomebrand.com) is pointed to that server.

Final Verification

A final check is done on the live site to ensure nothing broke during the move.

Deliverables

A live, publicly accessible website.
